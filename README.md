# Counter Application (React useState Project)

## 📌 Project Overview

The Counter Application is a simple React project developed to understand and practice the **useState** hook. It demonstrates how React manages component state and updates the user interface dynamically based on user interactions.

This project allows users to increment, decrement, and reset a counter while instantly reflecting state changes in the UI.

---

## 🎯 Objective

The objective of this project is to learn and implement React's **useState** hook for state management by building an interactive counter application.

---

## ✨ Features

- Increment counter value
- Decrement counter value
- Reset counter to zero
- Dynamic UI updates using React state
- Prevents counter from going below zero (optional)
- Displays **"Minimum limit reached"** when the counter reaches zero
- Simple and responsive user interface

## 📖 How It Works

- The application initializes the counter using the **useState** hook.
- Clicking **Increment** increases the counter by **1**.
- Clicking **Decrement** decreases the counter by **1**.
- The counter will not go below **0**.
- Clicking **Reset** sets the counter back to **0**.
- When the counter reaches **0**, a message **"Minimum limit reached"** is displayed.
